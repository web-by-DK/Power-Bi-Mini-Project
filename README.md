EdTech Course Catalog & Engagement Analysis
📌 Project Overview
This repository contains a comprehensive Power BI dashboard and data analysis project designed for an EdTech startup. The project analyzes a cleaned dataset of over 3,000 online courses to deliver actionable insights regarding content quality, viewer engagement, and category performance, directly guiding the startup's catalog expansion strategy.

🎯 Key Objectives
Establish Baselines: Calculate catalog scale, average course ratings, and total viewer engagement.

Category Analysis: Analyze demand across course types (Courses vs. Specializations) and subject categories.

Accessibility Impact: Evaluate the correlation between subtitle availability and viewership.

Instructor Identification: Highlight top-rated educators across key categories for future content partnerships.

📊 Dataset & Tech Stack
Dataset: Online_Courses.csv (Features include Category, Instructors, Rating, Viewers, Subtitles, and Duration).

Tools Used: Microsoft Power BI (Data Visualization, DAX, Power Query), Python (Pandas for data cleaning).

💡 Key Findings
Technical Dominance: Categories like Data Science, Information Technology, and Computer Science drive the highest average viewership per course.

Accessibility Drives Views: A strong positive correlation exists between the number of available subtitle languages and average viewer engagement, highlighting the value of localization.

Format Preferences: While standard courses generate the highest absolute views (7.0M), structured "Specializations" yield a higher average viewership per program.
